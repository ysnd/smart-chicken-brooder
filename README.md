# Smart Chicken Brooder

This project is an IoT based smart chicken brooder using telegram bot as a notification and control interface.

## Features
- Reads temperature and humidity.
- Send notification if temperature reach threshold value.
- Automatic heater control based on threshold value.
- Control feed servo, lamp, etc. using button in telegram bot. 

## Hardware Component
- ESP32/ESP8266
- DHT11
- Lamp 
- Relay
- Motor Servo
- some cardboard
